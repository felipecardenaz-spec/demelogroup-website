Project Name: DeMeloApps Website

Overview:
This is a high-end website for a software development company focused on SMBs and startups.

Goal:
- Showcase the company services and expertise
- Build trust and authority
- Convert visitors into leads
- Primary CTA: "Book a Free Call"

Target Audience:
- Startups
- Small and Medium Businesses (SMBs)
- Founders and business owners looking for custom software solutions

Positioning:
- Premium
- Modern
- High-performance
- Reliable tech partner

Design Direction:
- Dark theme
- Futuristic and tech-oriented
- Clean and minimal but visually striking

Visual Style:
- Linear-inspired UI (dark, sharp, modern)
- Apple-inspired glassmorphism (blur, transparency, depth)
- Subtle gradients and glow effects
- High contrast and readability

Key Features:
- Strong hero section with clear value proposition
- Smooth animations and transitions
- Interactive UI elements
- Animated buttons and hover states
- Tech-style background effects (glows, gradients, grid, noise, etc.)

Sections:
- Navbar (with CTA button)
- Hero (headline + subheadline + CTA)
- Social proof (logos / stats)
- Services / Features
- Process or workflow
- Testimonials (optional)
- Final CTA section
- Footer

Animation Level:
High:
- Smooth transitions
- Scroll animations
- Microinteractions
- Button hover effects
- Subtle motion everywhere

Tone:
- Confident
- Professional
- Modern
- Clear and direct

Performance Requirements:
- Fast loading
- Optimized animations
- Clean and efficient code

Success Criteria:
- Looks like a top-tier startup website
- Feels premium and modern
- Smooth and polished interactions
- Strong conversion towards booking a call
